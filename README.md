# infra
Репозиторій для розгортання інфраструктури

Для того щоб працювало спочатку проставити креди в файлі ~/.aws/credentials (Linux, Mac)

Далі

```
terraform init
```
```
terraform plan
```
```
terraform apply
```

Після завершення роботи видалити ресурси щоб не їло гроші
```
terraform destroy
```
